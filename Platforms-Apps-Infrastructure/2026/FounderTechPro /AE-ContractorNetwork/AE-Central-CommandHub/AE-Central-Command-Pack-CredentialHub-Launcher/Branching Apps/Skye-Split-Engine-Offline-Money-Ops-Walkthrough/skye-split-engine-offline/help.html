How to run the app well offline

1. Add contacts first
Save your common payout recipients or team members in Contacts so the split lane becomes faster.
2. Build payout math in Split Engine
Enter gross amount, deductions, choose a method, and save the scenario once the math looks correct.
3. Schedule recurring runs in Money Ops
Convert saved split plans into recurring payout templates, then push those runs into the deal ledger.
4. Issue receipts and protect the vault
Create settlement receipts after payouts, then keep JSON or encrypted backups outside the browser.

Important note

The app is fully static and has no external dependencies. It works by storing data in the browser. That means exported backups matter. For best resilience, keep both a local snapshot and an exported encrypted vault file.
