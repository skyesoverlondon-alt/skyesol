What is actually inside this upgrade?

Encrypted documents with tags and folders, a local attachment locker with preview/download/edit metadata flows, a device inventory lane, search across workstation content, snapshot retention, and a recovery bundle that includes the encrypted vault, snapshot history, and encrypted file store.

Documents that remain usable offline

Write procedures, runbooks, checklists, references, and internal docs in one local lane and export them when needed.

Actual file locker capability

Inventory that belongs in a workstation
