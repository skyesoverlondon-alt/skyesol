7-step field rhythm

Use this every time
01 · Dashboard
Open the dashboard first. Check vault health, storage usage, and whether your last encrypted backup is recent enough.
02 · Authenticator
Load accounts by paste, URI import, or batch import. Favorite hot accounts and use hide-codes mode when somebody is nearby.
03 · Contacts
Keep offline reach-out data, call notes, and field contacts in one place so you are not stranded when signal drops.
04 · Notes
Store sensitive operating notes locally. Pin high-priority items and export only when you choose to.
05 · Locker
Drop files into the locker for offline carry. Turn on PIN protection before loading anything you would hate to lose or expose.
06 · Backups
Run encrypted exports after meaningful changes. Plain JSON is inspection-friendly, but encrypted export is the serious recovery lane.
07 · Restore discipline
Create restore points before risky changes, imports, or cleanup passes. Test imports periodically so your backup is real, not theater.
Field rule
If it matters, give it both a device PIN and an encrypted export. That gives you on-device protection and portable recovery at the same time.
