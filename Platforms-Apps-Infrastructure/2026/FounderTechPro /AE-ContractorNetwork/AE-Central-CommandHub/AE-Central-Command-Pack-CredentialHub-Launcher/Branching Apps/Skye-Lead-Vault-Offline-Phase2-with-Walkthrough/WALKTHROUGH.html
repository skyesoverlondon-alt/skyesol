Offline operator guide

Skye Lead Vault Walkthrough

This is the fast working flow for the offline build. It is written so you can open the vault, get productive fast, and know exactly where the serious features live.

Fully offline Encrypted backup Quick capture Scripts + quotes Daily command center App lock
1. First launch

How to open it properly

1

Open index.html

That lands you on the Daily Command Center. If your browser supports installation, install it as an app for the cleanest offline use.

2

Go to Settings

Pick your background, glass feel, reminder preferences, and PIN. The vault lock is local to this device only.

3

Run your first backup

Before adding real data, go to Backup Vault and export one plain backup and one encrypted backup. That gives you a known-good restore point.

2. Daily use

The normal operating rhythm

A

Start on Dashboard

The dashboard is the working desk. It surfaces today’s follow-ups, hot leads, neglected records, overdue items, and local storage status.

B

Add new leads fast

Use + New lead for full entry or Quick capture when you are in the field and need speed.

C

Work stages from Pipeline

Move records through Prospect, Contacted, Warm, Proposal, Won, and Lost. Follow-up status keeps pressure visible.

3. Lead ledger

Where the real record lives

Lead Ledger is the master record area. Switch between table, cards, and dossier depending on how you want to work.

  • Smart dedupe: the vault checks business name, phone, email, and location similarity to flag duplicate risk.
  • Lead scoring: records get a local score from stage, priority, value, recency, due state, and manual boost.
  • Timeline: calls, notes, status changes, quote saves, and field captures stack into the activity history.
  • Document vault: keep lightweight files, photos, receipts, voice notes, and quote outputs attached to the lead.
  • Saved quotes: quote files saved from Playbook Studio can be stored back on the lead record.
4. Contact hub

Relationship mapping

Each business can hold multiple contacts, not just one. Use this to separate the owner, gatekeeper, manager, buyer, front desk, or referral source.

  • Mark decision-maker strength.
  • Store notes about best contact times and blockers.
  • Keep the business record clean while still knowing who matters.
5. Quick capture

Field mode

This is the speed lane. Use it when you are walking in, doing events, or moving too fast for full data entry.

  • Add a business with basic details and a quick note type.
  • Attach a photo when supported.
  • Record a voice note when supported by the browser/device.
  • It automatically creates a timeline event so the lead has context later.
6. Routes + territory

Offline planning without live maps

This lane is for route buckets and territory organization. You are not depending on map APIs here. You are grouping and sequencing work logically.

  • Create territory names like Central Phoenix, East Valley, West Run, Friday Route.
  • Sort leads by city, ZIP, area, or custom route tags.
  • Export printable run sheets for a day in the field.
7. Playbook studio

Scripts, offers, and quotes

This is where the vault turns records into action.

  • Script pack generator: creates reusable outreach language for a selected lead.
  • Copy or export: each block can be copied individually, or the full pack can be exported.
  • Quote builder: select a lead, choose a quote template, edit line items, and download an HTML quote.
  • Save to lead: store the quote back onto the lead so the dossier keeps history.
8. Analytics

What the numbers are telling you

Analytics is the local visibility layer. Use it to see whether you are actually moving work.

  • Stage distribution
  • Source mix
  • Recovery candidates
  • Open value estimates
  • Activity velocity
9. Backup vault

The safety flow

1

Export plain backup

Good for quick local copies and visible JSON inspection.

2

Export encrypted backup

Use a passphrase. This gives you AES-GCM encrypted backup JSON for safer storage outside the device.

3

Restore preview

Load a backup file first. The app previews what is inside before it changes anything.

4

Choose merge or replace

Merge combines current and incoming records by ID. Replace wipes the current vault and restores only the loaded file.

10. Privacy mode

How the lock works

The PIN lock is a local privacy barrier for this device and browser profile. It is useful protection for an offline field app, but it is not pretending to be enterprise identity.

  • Set or update the PIN in Settings.
  • Use Privacy mode or Lock now to hide the vault.
  • Unlock requires the correct local PIN.
Recommended workflow

Best way to use it day to day

Open Dashboard first. Knock out due follow-ups. Add any new field records through Quick Capture. Clean and enrich them in Lead Ledger. Use Contact Hub to map who matters. Build scripts and quotes in Playbook Studio. Review Analytics once a day. Export an encrypted backup before closing out real work.
