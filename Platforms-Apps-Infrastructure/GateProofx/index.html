Gate Proofx walkthrough

  1. Paste your admin bearer token (Data source) and click Save token.
  2. To load archives: click “List archive batches”, then Load a batch, or paste a key and click “Load from archive key”.
  3. To load exports: click Upload and select a gateway CSV/NDJSON export.
  4. Use Filters (text, level, provider, model, date range, status, max rows) to narrow results; charts and table update instantly.
  5. Sort columns by clicking headers; errors and warns are counted in stats.
  6. Export the current filtered view via “Export filtered CSV” or print/Save view as PDF.
  7. Charts: Requests over time, top providers, and levels doughnut reflect your current filters.

Tip: Keep tokens off the front-end by using the archive endpoint server-side in production; this UI uses your pasted token for convenience.
