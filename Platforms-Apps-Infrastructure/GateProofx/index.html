Gate Proofx — Full Walkthrough

Gate Proofx is the Gateway Data Explorer for kAIxU Gateway 13. It lets you load, filter, chart, and re-export every event that flowed through the gateway so you can audit usage, debug errors, and prove billing to clients.

1 — Load your data

  1. Upload a file — Click Upload CSV / NDJSON and pick an export file from the gateway admin’s Exports & Invoices tab. Supports .csv, .ndjson, .json, and .txt.
  2. Load from archive — Paste your admin bearer token into the Data-source section and click Save token. Then click List archive batches to browse archived NDJSON batches on Netlify Blobs, or paste a specific archive key and click Load from archive key.
  3. Your token is saved to localStorage for convenience. Keep it private.

2 — Filter

  1. Text contains — Free-text search across every field (provider, model, request_id, app_id, etc.).
  2. Level — Filter by info, warn, or error.
  3. Provider / Model — Narrow to a specific AI provider (openai, anthropic, gemini…) or model (gpt-4o, claude-3.5…).
  4. Date range — Set From / To timestamps to focus on a window.
  5. Status code — Show only a specific HTTP status (200, 429, 500…).
  6. Max rows — Cap the result set (10–5 000). Default 500.
  7. Charts and the data table update instantly as you change filters.

3 — Read the overview cards

  1. Rows — Total events in the current filtered set.
  2. Errors / Warns — Counts by log level for quick triage.
  3. Providers / Models — Distinct count of AI vendors and models hit.

4 — Charts

  1. Requests over time (line chart) — Shows daily request volume.
  2. Top providers (bar chart) — Which providers are handling the most traffic.
  3. Levels breakdown (doughnut) — Visual split of info / warn / error events.

5 — Data table

  1. Columns: Time, Level, Provider, Model, Status, Function, Request ID, App, Duration.
  2. Click any column header to sort (ascending / descending toggles).
  3. Rows are colour-neutral — hover to highlight.

6 — Export

  1. Export filtered CSV — Downloads the current filtered view as a .csv file you can open in Excel / Sheets.
  2. Save view as PDF — Opens the browser print dialog so you can save the entire page (charts included) as a PDF for client proof or internal records.

Why “Gate Proofx”?

The name comes from proof of gateway — you should be able to prove, with data, exactly what ran through the gateway, for whom, at what cost. This is your audit trail, billing proof, and debugging workbench — all in one page.

Typical workflow

StepAction
①Open Gate Proofx and paste your admin token.
②Click List archive batches → pick today’s batch → Load.
③Set Level = error to see failures. Check provider + status for patterns.
④Clear the filter. Set Provider = openai to view OpenAI-specific usage.
⑤Export as CSV for bookkeeping, or save PDF to share with a client.

Tip: Keep tokens off the front-end by using the archive endpoint server-side in production; this UI uses your pasted token for convenience.
