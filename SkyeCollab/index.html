    Tutorial

    Get the studio running with these steps.

    Step 1 · Store the key

    Save the `KAIXU_VIRTUAL_KEY` using the hero panel so all QA and Develop requests authenticate through kAIxuGateway13.

    Step 2 · QA prompt

    Type your QC question into the chat box and click "Ask QA". The kAIxu Gate v2 response streams into the chat thread with usage metadata.

    Step 3 · Develop stream

    Hit "Develop & Apply" to open the stream, watch `meta/delta/done` events update the timeline, and let the applying indicator show progress.
