Tutorial

Quickstart the compliance vault with these steps while keeping every request routed through kAIxuGateway13.

Step 1 · Store the Kaixu key

Use the panel above to store your `KAIXU_VIRTUAL_KEY`; this unlocks both the kAIxu Gate v2 and stream flows without exposing other secrets.

Step 2 · Classify

Drop a SkyeDocx snippet, pick a compliance category, and run the classification button. The chat endpoint returns a quick label plus usage metadata.

Step 3 · Stream metadata

Press "Stream metadata" to invoke kAIxu Gate v2 stream. Watch `meta`, `delta`, and `done` events fill the logs, timeline, and usage ledger.
