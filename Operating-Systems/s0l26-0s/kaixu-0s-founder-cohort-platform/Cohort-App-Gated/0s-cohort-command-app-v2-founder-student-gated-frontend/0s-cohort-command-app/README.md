# 0s Cohort Command Center

This package is the frontend-only cohort app with separated founder and student lanes.

Pages:
- index.html — launch surface
- founder-login.html — founder-only unlock
- founder-dashboard.html — founder board with student generator, roster, scoring, wiring map
- instructor-book.html — founder-only teaching book
- student-login.html — generated student ID entry
- student-portal.html — student-only workbook, notes, welcome letter, demo prep

This package persists locally in the browser for the current device and includes export/import for handoff. Backend wiring is intentionally kept in a dedicated founder-only settings surface.
