Founder lane and student lane separated cleanly

Run the cohort from one gated app without mixing teacher materials and student materials. ⚡

This front end is built around two completely separate class lanes: founder-only control surfaces and student-only class access. Founder tools generate live student IDs, welcome letters, roster records, attendance, final scoring, and cohort content. Students only see their own class lane after their generated ID is activated.
