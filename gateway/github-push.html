GitHub Push Console

This page is your dedicated GitHub Push entry point. Your tenant admin can link GitHub OAuth/PAT, and deployer/admin keys can push ZIP packages to connected repositories.

Open GitHub Push In Dashboard Open Full Dashboard

Setup Tutorial

  1. Log in with your Kaixu key.
  2. If your key role is admin/owner, connect GitHub OAuth or save a PAT.
  3. Select owner/repo/branch, upload a ZIP, and start push.
  4. Use job status rows to monitor attempts, retries, and commit URLs.

Billing Transparency

GitHub Push charges are tracked separately from AI usage but deducted under your same Kaixu key/customer balance by default, so one key can run the full platform.

All push jobs are tracked per customer and key for auditability and enterprise billing controls.
