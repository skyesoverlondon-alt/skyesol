Netlify Push Console

This page is your dedicated Netlify Push entry point. Your tenant admin can link Netlify credentials, map project IDs to site IDs, and deploy files from the dashboard using one Kaixu key.

Open Netlify Push In Dashboard Open Full Dashboard

Setup Tutorial

  1. Log in with your Kaixu key.
  2. If your key role is admin/owner, save your Netlify PAT in the Netlify Push tab.
  3. Add a project mapping: internal project ID plus Netlify site ID.
  4. Select files/folder and run push. Track status in the progress area and deploy URL output.

Billing Transparency

AI usage and Push usage are different charge lanes, but both are tracked under the same Kaixu key/customer balance by default. If you later want split-key billing (AI key vs Push-only key), that can be added as an opt-in policy change.

Everything is audited and usage-metered through gateway functions so charges remain traceable.
